# Simple WebSocket Chat App

This is a **real-time chat application** built using **WebSocket**, **Node.js**, **HTML**, **CSS**, and **JavaScript**.  
Multiple users can join the chat, send messages, and receive notifications when someone joins or leaves.

## Features

- Real-time messaging using WebSocket
- Multiple users can join simultaneously
- Join / Leave notification
- Simple UI
- Separate frontend and backend

## Technologies Used

- Node.js
- ws (WebSocket library)
- HTML
- CSS
- JavaScript
